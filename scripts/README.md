A small **Bash script** project to visualize disk usage of folders and files. Each item shows its **size** and a **proportional bar**.

---

## Features

- Shows **name and size** of files/folders  
- Displays **proportional bars**  
- Converts sizes to **B, K, M, G** automatically  
- Simple and easy to use  

---

## Usage

```bash
chmod +x diskVisualizer.sh
./diskVisualizer.sh /path/to/directory
